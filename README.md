Introduction
A simple command-line To-Do List application built with Python. This app allows users to add tasks, view tasks, mark tasks as done, delete tasks, and quit the app.

Features
Add tasks with a name
View all tasks with their status (done or not done)
Mark tasks as done
Delete tasks
Quit the app
Installation
To run this application, you need to have Python installed on your system. You can download the latest version of Python from the official Python website.

**Usage**
Clone this repository or download the Python file.
Open a terminal or command prompt and navigate to the directory where the Python file is located.
Run the Python file using the command python to_do_list_app.py.
Follow the menu prompts to add tasks, view tasks, mark tasks as done, delete tasks, or quit the app.
Functions
The following functions are used in this application:

add_task(name): Adds a new task with the given name.
view_tasks(): Displays all tasks with their status (done or not done).
mark_done(number): Marks a task as done based on the task number.
delete_task(number): Deletes a task based on the task number.
Example Use Cases
Add a new task: Choose option 1, enter the task name, and press Enter.
View all tasks: Choose option 2 to display all tasks with their status.
Mark a task as done: Choose option 3, view tasks, enter the task number, and press Enter.
Delete a task: Choose option 4, view tasks, enter the task number, and press Enter.
Note: This application does not store tasks persistently. All tasks are lost when the application is closed. If you want to store tasks persistently, you can modify the application to use a database or file storage.
